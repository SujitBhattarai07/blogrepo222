About The Blog

A simple place to write and share.

What is this?

The Blog is a simple blogging platform where anyone can sign up, write posts, and share their thoughts with others. No ads, no algorithms — just writing.

How it works

Create an account with a username and password. Once logged in, you can write a post and it will appear on the blog for everyone to read. You can delete your own posts at any time.

Built with

  • HTML, CSS, JavaScript — Frontend
  • Node.js — Backend API
  • PostgreSQL (Neon) — Database
  • Vercel — Hosting & Deployment
